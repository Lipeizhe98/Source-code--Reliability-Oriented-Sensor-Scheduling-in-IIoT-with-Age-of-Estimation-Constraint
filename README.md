# Source-code--Reliability-Oriented-Sensor-Scheduling-in-IIoT-with-Age-of-Estimation-Constraint
Simulation source code for the paper "Reliability-Oriented Sensor Scheduling in IIoT with Age of Estimation Constraint"

The code runs on a laptop equipped with an AMD Ryzen 7 5800H CPU, 16 GB of RAM, and a 512 GB SSD. MATLAB R2023a is required, and the Groubi solver must be installed.
